# FocusKit

A premium, minimalist, behavior-oriented digital ritual assistant designed to work seamlessly with physical FocusKit desk reset kits.

**Open less. Focus more.**

## What is FocusKit?

FocusKit is not a standalone Pomodoro utility. It's a physical-digital system: the mobile app initiates and supports rituals (a 25-minute focus session, phone parking, screen breaks, an evening shutdown ritual) that are designed to happen mostly *away* from the screen, alongside physical tools included in the FocusKit box — a Focus Planner, a 25-minute hourglass, and a Phone Parking pouch.

## Tiers

| Tier | Premium duration | Notes |
|---|---|---|
| Starter | 14 days | Free trial tier |
| Signature | 90 days (3 months) | Activated via QR/code in the Signature Desk Reset Kit |
| Corporate | 180–365 days | Team accounts; individual data never shown to employer |

## Core rituals

- **Focus Ritual** — a guided 25-minute focus session with phone parking and a visual hourglass timer
- **Daily Plan** — main focus, second focus, and one small win for the day
- **Weekly Plan** — top outcomes, things to reduce, one thing to protect
- **Shutdown Ritual** — an evening close-out: what you completed, what you'll start with tomorrow, what you're leaving behind
