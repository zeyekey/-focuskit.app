# Changelog

All notable changes to FocusKit will be documented in this file.

## [1.0.0] — 2026-06

### Added
- Initial release: Focus Ritual (25-minute guided session with phone parking and hourglass timer)
- Daily Plan and Weekly Plan
- Shutdown Ritual (evening close-out)
- QR / activation code system for Starter, Signature, and Corporate tiers
- Email, Google, and Apple sign-in
- Account deletion flow
- Push notifications for daily reminders and shutdown prompts
- Offline-first session tracking with cloud sync for premium accounts

### Known limitations
- Partial focus sessions (interrupted before completion) are not credited — only fully completed rituals count toward Progress
- Two devices used offline simultaneously may create duplicate session records (no merge/dedup yet)
